在回答或者实现与 Web 相关的问题时，请遵循以下规范：

1. **回答问题的方式**
   - 如果问题涉及代码实现或者错误修复，需要通过 Context7 查询最新的文档，以确保答案的准确性和时效性。

2. **代码风格要求**
   - 在使用 `import` 导入模块时，必须使用双引号 `"`，而不是单引号 `'`。

3. **其他注意事项**
   - 确保代码符合项目的编码规范。
   - 提供的解决方案应尽可能通用，适用于大多数 Web 项目。

4. **依赖管理工具**
   - 当前使用的依赖管理工具是 Yarn，而不是 npm。

5. **回答简洁性**
   - 避免创建过多的说明性 Markdown 文档，确保回答简洁明了，直接解决问题。

6. **执行 Yarn 命令的目录**
   - 所有与 Yarn 相关的命令必须在 `webapp` 目录下执行。
