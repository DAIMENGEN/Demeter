# 假期管理界面 (Holiday Management Page)

## 概述

假期管理界面提供了一个直观、高效的方式来管理全年的假期数据，包括法定假期和调休工作日。

## 功能特性

### 1. 年度日历视图
- 以年为单位展示全年12个月的日历
- 每个月以网格形式展示，清晰显示所有日期
- 响应式设计，支持多种屏幕尺寸

### 2. 日期类型区分
使用不同的颜色标识不同类型的日期：
- **法定假期**：红色背景 (#FEE2E2)
- **调休工作日**：蓝色背景 (#DBEAFE)
- **周末**：灰色背景 (#F3F4F6)
- **普通工作日**：白色背景
- **今天**：橙色边框突出显示

### 3. 假期操作

#### 添加假期
- **单个添加**：点击任意日期即可添加假期
- **批量添加**：
  1. 点击"批量选择"按钮进入批量模式
  2. 点击多个日期进行选择
  3. 点击"批量添加"按钮统一添加

#### 编辑假期
- 点击已标记为假期的日期
- 在弹出的表单中修改假期信息

#### 删除假期
- 鼠标悬停在假期日期上
- 点击右上角出现的删除图标
- 确认删除操作

### 4. 年份切换
支持三种方式切换年份：
- 下拉选择：当前年份 ± 5 年范围
- 上一年按钮：快速切换到前一年
- 下一年按钮：快速切换到后一年

### 5. 表单字段

添加/编辑假期时需要填写：
- **假期名称**（必填）：如"国庆节"、"春节"
- **类型**（必填）：
  - 法定假期 (holidayType = 1)
  - 调休工作日 (holidayType = 2)
- **日期**（必填，单个添加时）：选择具体日期
- **描述**（可选）：添加备注信息，最多200字符

## 技术实现

### 组件结构
```
holiday-page/
├── index.ts                          # 导出入口
├── holiday-page.tsx                   # 主页面组件
├── holiday-page.scss                  # 主页面样式
└── components/
    ├── year-calendar/                # 年度日历组件
    │   ├── index.ts
    │   ├── year-calendar.tsx
    │   └── year-calendar.scss
    └── holiday-modal/                # 假期编辑弹窗
        ├── index.ts
        ├── holiday-modal.tsx
        └── holiday-modal.scss
```

### 技术栈
- **React 19.2.0**: 使用函数组件和 Hooks
- **TypeScript 5.9.3**: 完整的类型安全
- **Ant Design**: UI 组件库
- **Day.js**: 日期处理
- **Sass**: 样式预处理

### 设计系统
遵循 UI/UX Pro Max 推荐的设计系统：
- **模式**: Data-Dense Dashboard
- **主色**: #7C3AED (紫色)
- **次色**: #A78BFA (浅紫色)
- **强调色**: #F97316 (橙色)
- **背景色**: #FAF5FF (浅紫色背景)
- **文字色**: #4C1D95 (深紫色)
- **字体**: Fira Code (代码) / Fira Sans (正文)

### 交互体验优化
1. **即点即编辑**: 点击日期立即弹出编辑表单
2. **悬停删除**: 鼠标悬停显示删除按钮，减少界面混乱
3. **批量操作**: 支持批量选择和添加，提高效率
4. **视觉反馈**: 
   - 悬停时日期边框高亮
   - 选中时显示明显的边框和阴影
   - 平滑的过渡动画 (150-200ms)
5. **响应式设计**: 
   - 桌面: 4列月份布局
   - 平板: 2-3列月份布局
   - 手机: 单列月份布局

### API 集成
使用已有的 Holiday API：
- `getAllHolidays()`: 获取指定年份的所有假期
- `createHoliday()`: 创建单个假期
- `batchCreateHolidays()`: 批量创建假期
- `updateHoliday()`: 更新假期信息
- `deleteHoliday()`: 删除假期

## 路由配置
页面路由: `/home/holiday`

## 使用示例

### 添加单个假期
1. 选择年份（如 2026）
2. 点击某个日期（如 10月1日）
3. 填写表单：
   - 假期名称: "国庆节"
   - 类型: "法定假期"
   - 描述: "国庆黄金周第一天"
4. 点击确定

### 批量添加假期
1. 点击"批量选择"按钮
2. 依次点击 10月1日-10月7日
3. 点击"批量添加 (7)"
4. 填写表单：
   - 假期名称: "国庆节"
   - 类型: "法定假期"
5. 点击确定

### 设置调休工作日
1. 点击周末日期（如10月12日，周六）
2. 填写表单：
   - 假期名称: "国庆调休"
   - 类型: "调休工作日"
3. 点击确定
4. 该日期将显示为蓝色工作日

## 可访问性
- 键盘导航支持
- 清晰的视觉层次
- 充足的色彩对比度 (4.5:1 以上)
- 响应式布局

- 仅在年份变化时重新加载数据
- 防止不必要的组件重渲染
